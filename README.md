Um gerenciador de tarefas funcional, inspirado no Trello, que permite organizar suas atividades de forma eficiente. Com uma interface intuitiva e responsiva, o site facilita o controle de tarefas, desde a criação até a conclusão.

---

##  Funcionalidades
- **Cadastro e Login de Usuários**: Autenticação segura com senhas criptografadas.
- **CRUD Completo para Tarefas**: Criação, leitura, atualização e exclusão de tarefas.
- **Interface Responsiva**: Desenvolvido com Bootstrap para compatibilidade com dispositivos móveis.
- **Organização Simples e Eficiente**: Gerencie suas tarefas com rapidez e praticidade.

---

##  Tecnologias Utilizadas

- **Linguagens de Programação**:
  - PHP
  - JavaScript
- **Frontend**:
  - HTML
  - CSS
  - Bootstrap
- **Banco de Dados**:
  - MySQL
- **Ambiente de Desenvolvimento**:
  - XAMPP (Apache e MySQL)
